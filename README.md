# 💎 ProFinance Ultimate (V17)

A powerful, privacy-focused, offline-first personal finance tracker built as a Single Page Application (SPA). No servers, no subscriptions, just pure financial management.

## ✨ Features
ProFinance combines the features of multiple paid apps into one lightweight file:

### 📊 Dashboard & Analytics
* **Visual Dashboard:** Real-time calculation of Net Balance, Income, Expense, and Savings.
* **Smart Charts:** Visual spending trends and category breakdowns (using Chart.js).
* **Calendar Heatmap:** Visualize your spending intensity day-by-day.
* **Top Merchants:** Automatically identifies where you spend the most money.

### 💰 Wealth & Planning
* **Net Worth Tracker:** Track assets vs. liabilities with a progress bar.
* **Forecasting:** AI-based prediction of your wealth in 1 year based on current habits.
* **Budgeting:** Set monthly limits per category with visual progress bars.
* **EMI Calculator:** Built-in loan amortization tool.
* **Tax Calculator:** Estimate tax based on progressive slabs or custom rates.

### 📝 Transaction Management
* **Smart Grouping:** Transactions are grouped by date with daily totals.
* **Smart Auto-Fill:** Typing "Uber" automatically categorizes it as "Transport."
* **Recurring Manager:** Auto-adds subscriptions when a new month starts.
* **Multi-Currency:** Support for local currency and USD conversion.
* **Search & Filter:** Instant search by note, tag, or category.

### ⚙️ Utilities
* **Privacy Mode:** One-click blur to hide sensitive numbers.
* **Dark/Light Theme:** Toggle between professional light mode and OLED-friendly dark mode.
* **Data Portability:** Full JSON Backup & Restore system.
* **Reports:** Generate professional PDF reports and CSV exports for Excel.
* **App-Like Experience:** Installable on iOS/Android as a PWA (Progressive Web App).

## 🚀 How to Use

### Option 1: Run Locally (Recommended)
1.  Download the `index.html` file.
2.  Double-click to open it in Chrome, Safari, or Edge.
3.  Start tracking!

### Option 2: Host it for Free
1.  Fork this repository.
2.  Enable **GitHub Pages** in the repository settings.
3.  Access your finance tracker from any device via the provided URL.

---

## 📱 Mobile Installation
ProFinance is designed to work like a native app.
* **iOS:** Open in Safari -> Tap 'Share' -> Tap 'Add to Home Screen.'
* **Android:** Open in Chrome -> Tap Menu -> Tap 'Install App.'

---

## 🔒 Privacy & Security
* **Offline First:** All data is stored in your browser's `localStorage`.
* **No Servers:** Your financial data never leaves your device.
* **No Tracking:** No analytics, no ads, no cookies.

---

## 🛠️ Tech Stack
* **Core:** HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript (ES6+).
* **Libraries (via CDN):**
    * Chart.js (for visualizations)
    * jsPDF (for PDF generation)
    * FontAwesome (for icons)
