# Expense Tracker

A simple web-based application for tracking daily income and expenses.
This project helps users monitor spending, analyze financial habits, and manage personal budgets using an easy dashboard interface.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0-green.svg)

---

## ✨ Features

### 📊 Dashboard & Analytics

* **Financial Dashboard:** Shows total balance, income, and expenses.
* **Charts & Graphs:** Visual representation of spending categories and trends.
* **Spending Analysis:** Helps identify where money is being spent the most.

### 💰 Finance Planning

* **Budget Management:** Set limits for categories and track spending.
* **EMI Calculator:** Estimate loan monthly payments.
* **Tax Calculator:** Calculate tax based on salary and rate.

### 📝 Transaction Management

* **Add / Delete Transactions**
* **Category Tracking:** Organize expenses like food, travel, shopping, etc.
* **Search & Filter:** Quickly find transactions using keywords.

### ⚙️ Extra Tools

* **Privacy Mode:** Hide financial values when needed.
* **Dark / Light Theme**
* **PDF & CSV Reports:** Export transaction reports.
* **Backup & Restore:** Save or load financial data easily.

## 🚀 How to Run the Project

### Run Locally

1. Download the project files.
2. Open `index.html` in any browser.
3. Start adding your expenses.

### Deploy on GitHub Pages

1. Upload the project files to a GitHub repository.
2. Go to **Settings → Pages**.
3. Select the **main branch** and save.
4. Your project will be available online.
